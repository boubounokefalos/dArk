# **dARK: A .ctf and .ARK theme for PSPs with the legendary ARK Custom Software.**


![HOME](https://raw.githubusercontent.com/boubounokefalos/dArk/refs/heads/main/pix/dARK_preview.jpg)

The Theme:
--------------
The purpose of the theme is to achieve a clean and minimal look with very few accent touches of teal.
I include both a .ctf and an .ARK theme in order to have a unified appearance either with the XMB or ARK launcher.
